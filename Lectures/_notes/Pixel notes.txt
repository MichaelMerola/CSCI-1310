pixel - the smallest unit of programmable color

# of pixels per area is related to the resolution

How to program a color:
	each pixel defined by color values
	red, green, blue weights
	0 - 255
	[255,0,0] all red
	[0,255,0] all green
	[0,0,255] all blue
	
Image
	Collection of pixels each location is a coordinate in the xy plane
	location defined by (x,y)
	Images defined by (x,y,[r,g,b])
